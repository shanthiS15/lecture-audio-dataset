# Lecture Audio Dataset

This dataset contains audio files used for lecture analysis.

## Dataset Details
• NPTEL Lecture audio clips (trimmed to 5 minutes)  
• TED Talk audio clips (maximum 20 minutes)  
• Format: WAV  
• Sample rate: 16kHz mono
